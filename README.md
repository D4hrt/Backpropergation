# Backpropagation Neural Network from Scratch

NumPy のみを用いて誤差逆伝播法 (Backpropagation) を実装した2層ニューラルネットワーク．2入力1出力の非線形関数 *f*(*x*, *y*) = sin(*x*) + cos(*y*) を学習・近似する．

## 概要

- **モデル**: 2層 MLP (入力 2 → 隠れ層 32 → 出力 1)
- **活性化関数**: 中間層 tanh，出力層 線形
- **損失関数**: 平均二乗誤差 (MSE)
- **最適化**: ミニバッチ SGD，勾配は誤差逆伝播法で計算
- **重み初期化**: Xavier 初期化 (tanh 向け)

ディープラーニングフレームワーク (PyTorch, TensorFlow 等) は使用せず，順伝播・逆伝播・パラメータ更新をすべてスクラッチで実装している．

## カスタマイズ

別の関数で実験する場合は，`target_function` を書き換える:

```python
def target_function(x, y):
    return np.sin(x) * np.cos(y)   # 例: 積の形
```

ハイパーパラメータ (隠れユニット数，学習率，エポック数，バッチサイズ) は `__main__` ブロック内で調整可能．

## 実装の要点

`TwoLayerNN.backward()` に誤差逆伝播法の中核処理が記述されている．出力層から入力層へ向かって，連鎖律により以下の順で勾配を計算する:

1. ∂*L*/∂ŷ : 出力に対する損失の勾配 (回帰のため (ŷ − *y*))
2. ∂*L*/∂*W*₂, ∂*L*/∂*b*₂ : 出力層のパラメータ勾配
3. ∂*L*/∂*a*₁ : 隠れ層活性化に対する勾配
4. ∂*L*/∂*z*₁ : 隠れ層 pre-activation に対する勾配 (tanh' を掛ける)
5. ∂*L*/∂*W*₁, ∂*L*/∂*b*₁ : 入力層側のパラメータ勾配
